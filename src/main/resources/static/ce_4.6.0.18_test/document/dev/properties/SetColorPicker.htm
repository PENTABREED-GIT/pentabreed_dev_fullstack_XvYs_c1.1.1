Param

Method

Event

 

 

SetColorPickerv4.3.1.03 or higher

색상 팔레트에 설정한 색상 값만 표시 및 확장 색상표 표시 여부를 설정합니다.

 

Parameters

객체명

용도

palette

2차원 배열로 색상 설정

extent

확장 색상표 표시 설정
- true : 확장 색상표 표시
- false : 확장 색상표 표시 안 함

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.SetColorPicker = {"palette":[["#3eb230","#444","#666","#999","#ccc"],
["#f00","#f90","#ff0","#0f0","#0ff"],["#f4cccc","#fce5cd","#fff2cc","#d9ead3","#d0e0e3"]], "extend":true};
CrossEditor.EditorStart();
