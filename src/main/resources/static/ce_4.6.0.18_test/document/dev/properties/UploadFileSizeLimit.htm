UploadFileSizeLimitv1.1 or higher

파일 업로드 시 업로드 파일의 용량을 제한합니다.

각각의 파일에 대한 용량 제한이 없을 경우 기본값은 "5MB"입니다.

 

입력 형식은 "파일 형식:용량"이며, 용량 단위는 "byte"입니다.

서로 다른 파일 형식에 대해 업로드 용량을 제한할 시에는 ",(콤마)"를 구분자로 입력하시면 됩니다.

 

각각의 파일 형식에 따라 용량을 다르게 제한할 수 있습니다.

만일, 이미지의 용량 제한(image:1048576)만을 설정했을 경우 이미지(image)는 1MB(1048576) 파일(file)은 5MB로 설정됩니다.

 

개발 언어가 ASP.NET인 경우 10MB 이상으로 설정 시 websource\aspx\web.config 파일의 maxRequestLength 설정값을 변경하셔야 합니다.
