CBInsertedImageExv4.1.1.03 or higher

에디터에 이미지 삽입된 후 callback 함수를 통해 삽입된 이미지 객체를 반환합니다.

 

또한, 어떤 방법으로 이미지가 삽입되었는지 정보 및

이미지/동영상/파일 링크 삽입 시 파일의 용량 정보와 파일 경로 정보도 함께 반환합니다.

 

다만, 이미지의 경우에만 객체 정보를 반환하며 그 이외에는 null 값이 리턴됩니다.

 

Parameters

객체명

용도

element

이미지 객체 (HTMLImageElement)

type

파일 삽입 타입
0 : 그림 넣기로 이미지 삽입
1 : Drag & Drop 삽입
2 : Copy & Paste 삽입 (업로드)
3 : Copy & Paste 삽입 (외부URL)

size

파일용량 (byte)

path

파일경로

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.event.CBInsertedImageEx = function(obj){
	console.log("element:" + obj.element);
	console.log("type:" + obj.type);
	console.log("size:" + obj.size);
	console.log("path:" + obj.path);
};
CrossEditor.EditorStart();
