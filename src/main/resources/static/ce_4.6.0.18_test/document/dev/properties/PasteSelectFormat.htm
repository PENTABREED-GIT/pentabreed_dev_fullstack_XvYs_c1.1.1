Param

Method

Event

 

 

PasteSelectFormatv4.3.2.21 or higher

오피스 플러그인 모드에서 붙여넣기 시, 붙여 넣는 데이터의 형태를 이미지 또는 HTML로 선택할 수 있도록 설정합니다.

단, 이미지 데이터가 클립보드에 포함되어 있는 경우에만 선택 창이 표시됩니다.

 

Parameters

TRUE

붙여넣기 선택 창 표시

FALSE

붙여넣기 선택 창 표시 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PasteSelectFormat = true;
CrossEditor.EditorStart();
