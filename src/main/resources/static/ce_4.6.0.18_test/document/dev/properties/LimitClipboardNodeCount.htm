LimitClipboardNodeCountv4.2.0.02 or higher

에디터에 붙여 넣는 데이터양을 설정합니다.

 

많은 양의 데이터를 복사&붙여넣기 시 브라우저가 중지되는 현상이 있어 복사한 데이터의 element node 개수가 1000개가 넘는 경우 메시지로 알리고 붙여넣기를 중지하도록 처리되고 있습니다.

 

단, Internet Explorer의 경우 클립보드에 직접 접근이 어려워 처리가 불가하여 반영되지 않습니다.

오피스 플러그인 모드의 경우에는 Internet Explorer도 지원됩니다.
