Param

Method

Event

 

 

ResponsiveUIv4.3.2.04 or higher

반응형 기본 설정 UI가 활성화된 상태로 에디터가 표시되도록 설정합니다.

기본으로 표시될 기기 설정도 가능합니다. (핸드폰, 태블릿, PC)

 

Parameters

TRUE

사용

FALSE

사용 안 함 (기본 값)

 

device : 'mobile'

핸드폰

device : 'tablet'

태블릿

device : 'pc'

PC

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ResponsiveUI = true;
CrossEditor.EditorStart();

 

var CrossEditor = new NamoSE("test");
CrossEditor.params.ResponsiveUI = {device : 'mobile'};
CrossEditor.EditorStart();
