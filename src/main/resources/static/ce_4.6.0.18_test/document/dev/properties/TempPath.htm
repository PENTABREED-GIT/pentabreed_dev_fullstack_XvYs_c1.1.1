TempPathv4.3.2.33 or higher

오피스 플러그인 기능 사용 시 임시 파일이 생성되는 폴더의 경로를 변경합니다.

설정한 폴더 아래에 CEWE 폴더가 생성됩니다.

 

단, 설정한 경로에 폴더 생성 권한이 없는 경우 기본 임시 폴더로 생성됩니다.

[ 기본 임시 폴더 ] C:\users\사용자\AppData\Local\Temp\CEWE

 

아래의 문자열 폴더명을 사용할 경우 사용자 기준의 경로를 설정 가능합니다.

 

사용자 기준 경로 설정 문자열 값

$APPDATA

C:\Users\사용자\AppData\Roaming

$LOCAL_APPDATA

C:\Users\사용자\Local

$MYDOCUMENTS

C:\Users\사용자\Documents

$PROGRAM_FILES

C:\Program Files (x86)

 

Parameters

설정

변경할 임시폴더 경로 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.TempPath = "D:\\temp";
CrossEditor.EditorStart();
