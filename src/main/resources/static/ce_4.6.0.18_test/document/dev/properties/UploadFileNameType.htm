Param

Method

Event

 

 

UploadFileNameTypev1.1 or higher

파일 업로드 시 파일명 저장 형식을 설정합니다.

 

개발 언어가 "PHP"인 경우, UploadFileNameType의 값을 'real'로 입력 시 에디터가 설치된 서버의 인코딩 type을 입력해 주어야 합니다.

 

Parameters

real

원본 파일의 파일명으로 저장 (기본 값)

trans

원본 파일의 파일명을 변환하여 저장 (Base64)

random
(1.5버전 이상)

원본 파일의 파일명을 변환하여 저장
(yyyyMMddHHmmssfff_Random문자 8자리)

 

Sample Codes

[ ASP, JSP, ASP.NET ]

var CrossEditor = new NamoSE("test");
CrossEditor.params.UploadFileNameType = "real";
CrossEditor.EditorStart();

 

[ PHP ]

var CrossEditor = new NamoSE("test");
CrossEditor.params.UploadFileNameType = "real,euc-kr";
CrossEditor.EditorStart();
