HTMLTabContents

HTML 탭에서 HTML 소스를 보여줄 때 HTML 소스 전체를 보여줄 것인지 BODY 태그의 내용만 보여줄지 설정합니다.

 

Parameters

BODY

BODY 태그내 내용만 보여주기 (기본 값)

HTML

HTML 소스 전체 보여주기

var CrossEditor = new NamoSE("test");
