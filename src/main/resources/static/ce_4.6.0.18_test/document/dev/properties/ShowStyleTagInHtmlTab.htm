ShowStyleTagInHtmlTabv4.3.2.17 or higher

Body 태그 내 내용만 보여주기 시 HTML 탭에서도 style 태그가 보이도록 설정합니다.

 

Parameters

TRUE

Body 태그 안에 style 위치

FALSE

Head 태그 안에 style 위치 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ShowStyleTagInHtmlTab = true;
CrossEditor.EditorStart();
