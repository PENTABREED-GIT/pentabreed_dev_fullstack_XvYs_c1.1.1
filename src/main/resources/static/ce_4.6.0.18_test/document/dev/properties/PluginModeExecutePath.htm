PluginModeExecutePathv4.3.1.05 or higher

별도의 이미지 서버 구성 시 오피스 플러그인에서 Copy&Paste 한 이미지를 업로드 실행할 URL 경로를 설정합니다.

 

같은 서버 내의 경로인 경우에는 크로스에디터 설치 폴더를 이용하여 설정이 가능합니다.

이미지 서버가 물리적으로 분리되어 있거나, 저장 경로가 URL로 접근할 수 없는 경우 이미지 서버도 웹 서비스가 되도록 설치해야 합니다.

 

Parameters

설정

이미지를 업로드 실행할 URL경로 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PluginModeExecutePath = "http://도메인/crosseditor/websource/Upload.do";
CrossEditor.EditorStart();
