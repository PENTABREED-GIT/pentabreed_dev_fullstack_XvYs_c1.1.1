CreateToolbar

도구 막대(Toolbar)의 구성요소를 설정합니다.

 

CreateToolbar는 UserToolbar의 설정값이 true로 설정되어야 사용 가능합니다.

각각의 구성요소는 파이프(|)를 구분자로 하여 입력하시면 됩니다.

각각의 구성요소 키워드를 보시려면 [ 여기 ]를 클릭하십시오.

 

UserToolbar의 설정값이 true로 설정되어 있더라도 CreateToolbar의 값이 없을 경우, 기본적으로 정의된 구성요소에 의해 도구막대가 생성됩니다.

 

Parameters

설정

설정한 툴바만 보여짐

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UserToolbar = true;
CrossEditor.params.CreateToolbar = "newdoc|copy";
CrossEditor.EditorStart();
