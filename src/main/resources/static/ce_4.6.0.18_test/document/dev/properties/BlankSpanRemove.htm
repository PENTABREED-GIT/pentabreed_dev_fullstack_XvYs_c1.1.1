Method

Event

 

 

BlankSpanRemovev3.5.0.16 or higher

빈 span 태그를 제거할 수 있는 설정입니다.

 

Internet Explorer 11버전의 경우 특정 소스에서 &nbsp; 뒤에 한글 입력이 정상적으로 되지 않는 현상이 발생하여, 버그를 일으키는 빈 span 태그를 모두 제거합니다.

- 적용 범위 : 탭 이동, SetValue, SetBodyValue, Copy&Paste

 

Parameters

TRUE

빈 span태그 제거

FALSE

빈 span태그 제거 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.BlankSpanRemove = true;
CrossEditor.EditorStart();
