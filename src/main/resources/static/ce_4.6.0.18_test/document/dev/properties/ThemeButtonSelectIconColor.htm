Param

Method

Event

 

 

ThemeButtonSelectIconColorv4.5.0.05 or higher

선택된 도구 단추(Toolbutton)의 색상을 설정합니다.

단, NewToolbar를 설정한 경우에만 적용이 가능합니다.

 

Parameters

설정

선택된 색상 설정

예) "#FF0000" or "blue"

설정 안 함

기본 값 (#666666)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.NewToolbar = true ;
CrossEditor.params.ThemeButtonSelectIconColor = "#FF0000";
CrossEditor.EditorStart();
