Event

 

 

DocBaseURL

에디터 로딩 시 또는 '새 문서' 명령을 실행할 경우 기본 서식으로 사용할 html 파일의 URL 경로를 설정합니다.

 

html 파일 경로는 전체 경로나 절대 경로를 사용합니다.

에디터가 실행되는 서버의 도메인 정보와 다른 도메인의 파일 경로는 사용할 수 없습니다.

값이 설정되지 않은 경우 기본 편집(Edit) 창으로 설정됩니다.

 

Parameters

설정

기본 서식파일의 경로를 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.DocBaseURL = "/ce/newdoc.html";
CrossEditor.EditorStart();
