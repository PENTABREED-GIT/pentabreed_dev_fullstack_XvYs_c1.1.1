Param

Method

Event

 

 

EditorBaseURLv1.1 or higher

현재 설치된 에디터의 기본 URL 경로를 설정합니다.

 

EditorBaseURL의 입력 값은 에디터 설치 폴더에 접근할 수 있는 Full URL을 입력합니다.

단, EditorBaseURL은 에디터가 실행되는 도메인과 같은 도메인을 사용해야 에디터가 정상 동작합니다.

 

Parameters

설정

에디터의 기본 URL 경로 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.EditorBaseURL = "https://comp.namoeditor.co.kr/crosseditor";
CrossEditor.EditorStart();
