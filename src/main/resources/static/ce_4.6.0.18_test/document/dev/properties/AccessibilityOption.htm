AccessibilityOption

사용자가 콘텐츠를 입력할 때 적용할 웹 접근성 준수 정도를 설정합니다.

잘못된 값을 지정하거나 값을 지정하지 않은 경우 사용 안 함으로 설정됩니다.

 

Parameters

0

사용 안 함 (기본 값)

1

1단계 (웹 접근성 기능 필수)

2

2단계 (1단계 + 외부 입력 불가)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.AccessibilityOption = 1;
CrossEditor.EditorStart();
