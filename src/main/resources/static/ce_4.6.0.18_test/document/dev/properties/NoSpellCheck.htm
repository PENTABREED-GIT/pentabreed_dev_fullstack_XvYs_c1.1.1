NoSpellCheckv3.5.1.02 or higher

크롬 브라우저의 SpellCheck 속성을 해제합니다.

맞춤법 검사 기능은 브라우저 자체 기능으로 활성화된 경우 에디터 본문에 빨간 밑줄이 표시됩니다.

 

많은 양의 데이터를 에디터로 붙여넣기 한 경우 크롬 브라우저에서 맞춤법 검사 수행 시 브라우저가 느려지는 현상이 있어 SpellCheck 속성을 해제하도록 설정합니다.

 

맞춤법 기능 해제를 원하시는 경우 아래의 방법으로 설정이 가능합니다.

IE

[인터넷옵션] – [프로그램] – [추가기능 관리] – 왼쪽메뉴 중
"맞춤법 교정" 메뉴선택 후 하단의 " 맞춤법 교정 사용" 체크 해제

Chrome

[설정] – [고급] – [언어] – "맞춤법 검사" 기능 해제

NoSpellCheck API를 설정하여 해제

 

Parameters

TRUE

SpellCheck 해제

FALSE

SpellCheck 사용 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.NoSpellCheck = true;
CrossEditor.EditorStart();
