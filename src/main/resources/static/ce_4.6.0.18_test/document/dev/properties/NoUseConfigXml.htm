NoUseConfigXmlv3.5.1.12 or higher

에디터 설정 파일(Config.xml)과 통신하지 않고 에디터가 실행될 수 있도록 설정합니다.

보안 취약점 사항 중 설정 파일 정보가 노출되는 현상으로 에디터 설정 정보를 객체 형태로 선언하여 설정합니다.

또한, 에디터 로딩 시 통신하는 속도를 줄이고자 할 때 설정합니다.

 

params로 설정한 API는 객체로 설정이 가능하며 "API 명" : "설정값" 형태로 설정합니다

아래의 필수 설정값이 기본으로 설정되어야 하며 필수 설정값이 없는 경우 에디터가 로딩되지 않습니다.

 

[ 필수 설정 ]

제품 이름

"Name" : "Namo CrossEditor"

서버 언어

"WebLanguage" : "JSP" (신청한 서버 언어 정보)

HTML탭 소스 출력 설정

"HTMLTabContents" : "body"

스킨 설정

"Skin" : "default"

 

Parameters

TRUE

Config.xml 파일과 통신 안 함

FALSE

Config.xml 파일과 통신 (기본 값)

 

Sample Codes

var paramObj = {
    "Name" : "Namo CrossEditor",
    "WebLanguage" : "JSP",
    "HTMLTabContents" : "body",
    "Skin" : "default",
    "NoUseConfigXml" : "true",
    ...	
    추가 설정
};

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart(paramObj);

function OnInitCompleted(e){
    CrossEditor.SetBodyValue(document.getElementById("contents").value);
}
