Fontv2.0.2.16 or higher

도구 막대에 표시되는 글꼴 목록을 설정합니다.

 

글꼴 목록이 재정의 되어 설정됩니다.

글꼴 추가만 원하시는 경우에는 {기본 글꼴 목록 + 추가 글꼴} 형태로 설정해 주셔야 합니다.

 

설정값은 아래와 같은 형태로 입력하시면 됩니다.

{"실제 style에 들어갈 이름" : "SelectBox에 표시할 이름"}

 

단, 클라이언트 PC에 설정한 글꼴이 설치되어 있지 않은 경우에는 기본 글꼴로 보입니다.

 

기본 글꼴 목록

{"맑은 고딕":"맑은 고딕", "Dotum":"돋움", "Gulim":"굴림", "Batang":"바탕", "Gungsuh":"궁서", "David":"David", "MS PGothic":"MS PGothic", "New MingLiu":"New MingLiu", "Simplified Arabic":"Simplified Arabic", "simsun":"simsun", "Arial":"Arial", "Courier New":"Courier New", "Tahoma":"Tahoma", "Times New Roman":"Times New Roman", "Verdana":"Verdana"}
