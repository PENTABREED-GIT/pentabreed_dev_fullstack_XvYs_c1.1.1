Method

Event

 

 

AlertCopyPasteImgv2.0.5.01 or higher

편집 창에 붙여 넣기 시 로컬 경로 이미지 포함 여부를 체크하여 경고 메시지가 표시되도록 설정합니다.

[ 삽입된 데이터 중 로컬경로 이미지는 지원되지 않습니다. ]

 

※ 3.5버전 부터는 이미지 붙여넣기가 지원되어 해당 API를 설정하여도 동작하지 않습니다.

 

Parameters

TRUE

경고메시지 설정

FALSE

경고메시지 설정 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.AlertCopyPasteImg = true;
CrossEditor.EditorStart();
