Param

Method

Event

 

 

PluginModeSetupPathv4.3.4.02 or higher

오피스 플러그인 설치 파일의 설치 경로를 설정합니다.

설치 경로만 설정 시에는 Setup.exe 또는 Setup_en.exe 파일을 설치하며 파일까지 설정이 가능합니다.

 

단, 동일 도메인 또는 CORS(교차 출처 공유)가 허용되는 URL만 설정이 가능합니다.

 

Parameters

설정

설치 경로 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PluginModeSetupPath= "https://comp.namoeditor.co.kr/crosseditor/ceplugins/";
CrossEditor.EditorStart();
