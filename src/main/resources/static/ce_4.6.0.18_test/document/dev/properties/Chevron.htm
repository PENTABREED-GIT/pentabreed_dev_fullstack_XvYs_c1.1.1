Param

Method

Event

 

 

Chevronv4.3.2.01 or higher

에디터 너비가 도구막대의 너비보다 작을 경우 도구막대 우측에 축약된 드롭다운 메뉴를 표시합니다.

 

툴바가 개행 되면서 줄바꿈이 되는 경우 에디터 높이가 정상적으로 계산되지 못하기 때문에, 도구막대 너비가 에디터 너비를 벗어나는 경우 Chevron 설정을 하는 것을 권장합니다.

 

Parameters

TRUE

사용

FALSE

사용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.Chevron = true;
CrossEditor.EditorStart();
