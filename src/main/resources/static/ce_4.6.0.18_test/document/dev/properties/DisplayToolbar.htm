DisplayToolbarv4.3.2.22 or higher

에디터 로딩 시 도구막대(Toolbar)가 표시되지 않도록 설정합니다.

 

ShowToolbar 설정 후 에디터 로딩 시 툴바가 감춰지는 과정이 보이지 않도록 설정합니다.

모던 웹 브라우저에서만 지원이 가능합니다.

 

Parameters

TRUE

툴바 표시 (기본 값)

FALSE

툴바 감추기

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.DisplayToolbar = true;
CrossEditor.EditorStart();
