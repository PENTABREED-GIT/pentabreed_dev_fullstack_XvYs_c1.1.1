UseEnhancedImageInsertv4.3.2.01 or higher

그림 넣기로 이미지 삽입 시 미리 보기 화면을 통하여 원본을 회전/반전할 수 있는 기능을 설정합니다.

그림 넣기 실행 시 이미지 선택 창이 나타나고 이미지 선택 시 미리 보기 화면에는 회전/반전 적용이 가능합니다.

 

모바일의 경우 이미지의 용량이 크기 때문에 이미지 품질을 조정하여 업로드할 수 있도록 EnhancedImageInsertUploadQuality API도 제공하며 함께 설정해 주시는 것을 권장합니다.

 

Parameters

TRUE

사용

FALSE

사용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UseEnhancedImageInsert = true;
CrossEditor.params.EnhancedImageInsertUploadQuality = 0.6;
CrossEditor.EditorStart();
