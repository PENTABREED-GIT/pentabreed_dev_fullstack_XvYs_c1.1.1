Param

Method

Event

 

 

CBInsertedImagev4.0.1.04 or higher

에디터에 이미지가 삽입된 후 callback 함수를 통해 삽입된 이미지 객체를 반환합니다.

또한, 어떤 방법으로 이미지가 삽입되었는지 정보도 함께 반환합니다.

 

Parameters

객체명

용도

imgobj

이미지 객체 (HTMLImageElement)

insertype

이미지 삽입 타입
0 : 그림 넣기로 이미지 삽입
1 : Drag & Drop 이미지 삽입
2 : Copy & Paste 이미지 삽입 (업로드)
3 : Copy & Paste 이미지 삽입 (외부URL)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.event.CBInsertedImage = function(imgobj, insertype){
	var orgsrc = imgobj.src;
	var type = insertype;
	
	var newimg = imgorg.ownerDocument.createElement("img");
	newimg.src = "https://comp.namoeditor.co.kr/ce4/help/ko/images/title_ci.png";
	
	imgobj.parentNode.insertBefore(newimg, imgobj);
	imgobj.parentNode.removeChild(imgobj);
};
CrossEditor.EditorStart();
