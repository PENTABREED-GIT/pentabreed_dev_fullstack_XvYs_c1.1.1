LineHeight

에디터 편집 창에서 텍스트를 입력할 때 문단 내 기본 줄 간격을 설정합니다.

 

입력 값은 long 타입으로 입력하며, 단위는 퍼센트(%)입니다.

값을 설정하지 않는 경우, 기본 값은 '120%'으로 설정됩니다.
