Method

Event

 

 

SetDebug

예외사항이나 오류 발생 시 오류 메시지 출력 여부를 설정합니다.

설정 시 경고 창이 화면에 나타납니다.

 

오류 메시지가 나타나도록 설정한 경우 관리자 페이지의 미리 보기에서만 확인할 수 있습니다.

 

Parameters

TRUE

실행

FALSE

실행 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.SetDebug = true;
CrossEditor.EditorStart();
