Method

Event

 

 

Height

에디터의 전체 높이를 설정합니다.

 

높이의 입력은 long 타입으로 입력하며, 단위는 픽셀(pixel)입니다.

Height 값을 설정하지 않는 경우, 기본 값은 '300px'으로 설정됩니다.

 

에디터의 최소 높이 값은 '300px'이며 300px 이하로 설정하여도 반영되지 않습니다.

설정

var CrossEditor = new NamoSE("test");
CrossEditor.params.Height = 500;
CrossEditor.EditorStart();
