UseTagCleanerExv4.3.3.03 or higher

태그 정리 기능 동작 시 로딩 바 표시 여부를 설정합니다.

에디터에 값을 Set 한 이후 태그 정리 기능을 수행하면서 로딩 바가 표시됩니다.

 

Parameters

TRUE

로딩바 표시

FALSE

로딩바 표시 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UseTagCleanerEx = true;
CrossEditor.EditorStart();
