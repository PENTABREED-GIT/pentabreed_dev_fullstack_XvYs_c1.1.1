Param

Method

Event

 

 

ShowRulerv4.1.1.05 or higher

눈금자 기능 표시 여부를 설정합니다.

가로 눈금자, 세로 눈금자 각각 표시하도록 설정도 가능합니다.

 

Parameters

TRUE

표시

FALSE

표시 안 함 (기본 값)

 

horizontal

true

가로 눈금자 표시

false

가로 눈금자 표시 안 함

vertical

true

세로 눈금자 표시

false

세로 눈금자 표시 안 함

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ShowRuler = true;
CrossEditor.EditorStart();

 

var CrossEditor = new NamoSE("test");
CrossEditor.params.ShowRuler = {horizontal : true, vertical : false};
CrossEditor.EditorStart();
