Param

Method

Event

 

 

ServerUrlv4.0 or higher

에디터에서 생성되는 기본 URL 정보의 도메인 정보 포함 여부를 설정합니다.

설정값을 2로 설정했다면 자신의 도메인 정보를 제외하고 표시됩니다.

 

Parameters

1

도메인 주소를 포함하는 전체 경로 (기본 값)

2

도메인 주소를 제외한 절대 경로

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ServerUrl = 2;
CrossEditor.EditorStart();
