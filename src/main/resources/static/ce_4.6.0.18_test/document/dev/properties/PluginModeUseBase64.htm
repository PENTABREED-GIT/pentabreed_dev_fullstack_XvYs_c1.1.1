Event

 

 

PluginModeUseBase64v4.3.3.01 or higher

오피스 플러그인 모드에서 이미지 Copy&Paste 업로드 시 base64string으로 변환한 문자열을 JSON 형태로 서버에 전송합니다. (모든 브라우저)

 

PluginModeUseBase64 API는 4.3.3.01 버전부터 설정이 가능하며, 4.3.2.35 이하 버전에서 오피스 플러그인을 사용하고 있었다면 4.3.3.01 이상 버전 업데이트 시 해당 API 설정이 필요합니다.

 

Parameters

TRUE

base64string 방식

FALSE

multipart/form 방식 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PluginModeUseBase64 = true;
CrossEditor.EditorStart();
