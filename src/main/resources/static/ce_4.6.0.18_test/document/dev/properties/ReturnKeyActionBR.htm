ReturnKeyActionBRv2.0.2.26 or higher

Enter 키를 이용하여 개행 시 P 태그를 BR 태그로 설정합니다.

 

설정할 경우 BR 태그는 단순 줄바꿈 태그이므로 문단 태그에 설정되는 정렬/들여 쓰기/내어 쓰기는 모두 한 단위로 적용되어 움직입니다.

Internet Explorer11 브라우저에서 BR 태그가 삽입되는 경우 특정 위치에서 한글 입력 시 자음/모음이 분리되거나, 줄바꿈이 되었다가 다시 윗줄로 커서가 이동하는 현상이 발생할 수 있습니다. (브라우저 버그사항)

 

기본 P 태그로 설정하여 사용하는 것을 권장합니다.

 

Parameters

TRUE

br태그로 설정

FALSE

p태그로 설정 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ReturnKeyActionBR = true;
CrossEditor.EditorStart();
