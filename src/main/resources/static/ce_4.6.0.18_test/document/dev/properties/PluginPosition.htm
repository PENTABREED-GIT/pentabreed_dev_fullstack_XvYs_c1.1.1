PluginPositionv4.0.0.16 or higher

iframe 안에 에디터 생성 시 iframe 노드를 설정하여 플러그인 창 위치를 설정합니다.

 

Parameters

TRUE

플러그인 창이 위치할 iframe 노드 설정

FALSE

window.top (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PluginPosition = $(window.top.document.body).find('iframe#POPPanel').get(0);
CrossEditor.EditorStart();
