MimeEncodeIncludeUrlv4.5.0.01 or higher

MimeEncodeFileScope가 Local인 경우, MIME 인코딩에 포함시킬 원격 파일의 시작 URL을 설정합니다.

URL을 설정할 때 프로토콜(ftp, http 등) 및 절대 경로와 비교하므로 반드시 절대 경로를 설정해야 합니다.

 

URL의 처음 부분만 비교를 하므로 아래의 경우 다른 결과를 가져올 수 있습니다.

MimeEncodeIncludeUrl

동작

http://www.aabbcc.co.kr/image

설정된 URL로 시작되는 모든 리소스가 포함
예) http://www.aabbcc.co.kr/image_news.gif 등

http://www.aabbcc.co.kr/image/

해당 폴더 이하의 리소스가 포함
예) http://www.aabbcc.co.kr/image/news/123456.gif 등

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MimeEncodeIncludeUrl = "http://www.aabbcc.co.kr/image/";
CrossEditor.EditorStart();
