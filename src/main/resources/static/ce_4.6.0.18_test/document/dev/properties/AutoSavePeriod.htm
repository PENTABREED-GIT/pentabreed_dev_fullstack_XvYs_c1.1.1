AutoSavePeriodv4.0.1.07 or higher

자동 저장 기능 활성화 여부 설정 및 저장시간을 설정합니다.

 

설정값은 분 단위이며, 브라우저의 로컬 스토리지에 데이터가 저장됩니다.

0으로 설정 시 에디터 [환경설정]에서 '자동 저장 사용' 기능이 비활성화되어 표시됩니다.

Internet Explorer 브라우저의 경우 10버전 이상에서 지원이 가능합니다.

 

최대 20개가 저장되며 20개가 넘는 경우 덮어쓰기 합니다.

1시간 이후의 데이터는 삭제되며 1시간이 넘은 데이터는 새로운 데이터 생성 시 모두 삭제됩니다.

 

FullURL 기준으로 페이지의 내용이 저장됩니다.

예를 들어, 아래와 같은 주소인 경우 브라우저에서는 같은 페이지로 표시되지만 실제 href 값이 다르기 때문에 저장 공간을 다르게 사용합니다.

  1) ce.com/

  2) ce.com/index.html

  3) ce.com/index.html?

 

Parameters

0

자동저장 사용 안 함

1

1분 단위로 자동저장 (기본 값)
