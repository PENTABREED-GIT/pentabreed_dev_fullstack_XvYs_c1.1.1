ParagraphTagStylev4.0.1.04 or higher

설정한 스타일을 에디터에서 값 호출 시 문단 태그 inline style로 설정합니다.

 

GetValue / GetBodyValue 함수 호출 시 적용됩니다.

에디터 작성 시

GetValue/GetBodyValue 함수 호출 시

<style type="text/css" id="NamoSE__ParagraphTagStyle">
p:not(.ce_exstyle){font-size:20pt; font-family:궁서체;}
</style>
<p>나모</p>

<p style="font-size: 20pt; font-family: 궁서체;">나모</p>

 

SetValue / SetBodyValue 함수를 통하여 값을 넣는 경우 기존 본문을 구분 지을 수 있는 P 태그 (id=ce_split)를 중간에 삽입하면 삽입한 태그 이후의 값은 ParagraphTagStyle에 설정한 스타일의 영향을 받지 않습니다.

예시

<p><br /></p>              → ParagraphTagStyle 영향 받음

<p id='ce_split'></p>     → 삭제됨

<p>이전 데이터 본문</p> → ParagraphTagStyle 영향 받지 않음

 

Parameters

설정

문단 태그에 설정할 스타일 값

{"스타일 속성명" : "스타일 속성값"}

설정 안 함

var CrossEditor = new NamoSE("test");
CrossEditor.params.ParagraphTagStyle = {"font-size":"20pt", "font-family":"궁서체"};
CrossEditor.EditorStart();

function OnInitCompleted(e){
   var contents = "<p><br /></p><p id='ce_split'></p>" + document.getElementById("data").value;
   CrossEditor.SetBodyValue(contents);
}
