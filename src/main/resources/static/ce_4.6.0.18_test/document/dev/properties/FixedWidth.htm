FixedWidthv4.1.1.05 or higher

설정한 너비로 페이지를 표시합니다.

Internet Explorer 브라우저의 경우 10버전 이상에서 지원이 가능합니다.

 

너비의 입력은 long 타입으로 입력하며, 단위는 픽셀(pixel)입니다.
