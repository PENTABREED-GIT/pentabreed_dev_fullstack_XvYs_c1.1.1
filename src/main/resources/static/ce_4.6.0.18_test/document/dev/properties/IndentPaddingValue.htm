IndentPaddingValuev3.0 or higher

에디터 편집 창에서 텍스트를 입력할 때의 문단 들여쓰기 간격을 설정합니다.

입력 값은 long 타입으로 입력하며, 단위는 픽셀(pixel)입니다.

 

들여쓰기 시 margin-left 값이 설정됩니다.
