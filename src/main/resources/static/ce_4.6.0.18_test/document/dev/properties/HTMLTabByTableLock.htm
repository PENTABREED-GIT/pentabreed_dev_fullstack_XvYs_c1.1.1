Param

Method

Event

 

 

HTMLTabByTableLockv1.5 or higher

현재 문서 내에 표 보호 기능이 동작하는 표가 있을 경우 HTML 탭 사용 여부를 설정합니다.

표 보호 기능의 경우 관리자 용도(Manager)로 계약된 경우에만 기능이 제공됩니다.

 

Parameters

TRUE

HTML탭 사용

FALSE

HTML탭 사용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.HTMLTabByTableLock = true;
CrossEditor.EditorStart();
