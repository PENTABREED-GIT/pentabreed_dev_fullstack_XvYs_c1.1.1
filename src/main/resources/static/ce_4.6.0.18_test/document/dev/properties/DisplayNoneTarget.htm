DisplayNoneTargetv1.1 higher

에디터 최초 실행 시 화면에 바로 표시하지 않는 경우(display:none) 에디터를 표시할 Element를 설정합니다.

에디터를 표시할 Element가 설정되어 있지 않은 경우에는 에디터 높이를 계산하지 못하여 정상적으로 화면에 표시되지 않습니다.

 

Parameters

설정

에디터를 감싸고 있는 Element 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.DisplayNoneTarget = document.getElementById("crosseditor");
CrossEditor.EditorStart();
