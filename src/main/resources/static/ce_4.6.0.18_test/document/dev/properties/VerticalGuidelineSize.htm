VerticalGuidelineSizev4.3.2.04 or higher

설정한 값의 위치에 세로 구분선이 빨간 점선으로 표시되도록 설정합니다.

모던 웹 브라우저에서 지원이 가능하며 Internet Explorer 10 이하에서는 구분선이 출력되지 않습니다.

 

에디터는 페이지 개념이 없기 때문에 웹 페이지 마크업 단위인 픽셀(px)로 설정이 가능합니다.
