RemoveScriptBlockv4.1.1.05 or higher

본문 내 Script Block 삭제 여부를 설정합니다.

설정하면 Script Block 전체를 탭 이동 시, 작성된 내용 리턴(GetValue, GetBodyValue) 시 제거합니다.

 

Parameters

TRUE

스크립트 허용 안 함

FALSE

스크립트 허용 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.RemoveScriptBlock = true;
CrossEditor.EditorStart();
