MimeEncodeTypev4.5.0.01 or higher

MIME 형식으로 데이터를 얻을 때에 인코딩할 방법을 설정합니다.

 

Parameters

설정

인코딩 방법

0

인코딩하지 않음 (영문 7bit, 한글 8bit)

1

7bit가 아닌 경우에만 base64 인코딩 (기본 값)

2

7bit가 아닌 경우에만 quoted-printable 인코딩

3

무조건 base64 인코딩

4

무조건 quoted-printable 인코딩

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MimeEncodeType = 1;
CrossEditor.EditorStart();
