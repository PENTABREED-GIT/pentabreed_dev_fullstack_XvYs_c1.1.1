UseTagCleanerv4.3.2.23 or higher

태그 정리 기능을 이용하여 설정에 따라 태그 사이의 공백 제거 및 중첩된 span 태그를 병합합니다.

 

ture 설정 시 GetValue, GetBodyValue, SetValue, SetBodyValue, 툴바/메뉴바에서 태그 정리 기능 실행 시 동작합니다. false 설정 시에는 툴바/메뉴바에서 태그 정리 기능 실행 시에만 동작합니다.
