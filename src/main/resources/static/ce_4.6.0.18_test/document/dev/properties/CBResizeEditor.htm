CBResizeEditorv4.2.0.08 or higher

에디터의 ResizeBar 실행 시 callback 함수를 통해 에디터의 너비, 높이 정보를 반환합니다.

 

Parameters

객체명

용도

width

에디터 너비

height

에디터 높이

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.event.CBResizeEditor = function(obj){
	console.log("width:" + obj.width);
	console.log("height:" + obj.height);
};
CrossEditor.EditorStart();
