CreateTabv1.5 or higher

화면에 표시할 탭의 종류를 설정합니다.

 

설정값은 파이프(|)를 구분자로 하여 입력하시면 됩니다.

예를 들어, 편집 탭과 HTML 탭을 화면에 표시하는 경우 "0|1"과 같은 형식으로 설정합니다.

 

편집 탭은 화면에 항상 표시되며 설정값 이외의 값을 입력하였을 경우 편집 탭만 화면에 표시됩니다.

단, 아무 값도 입력을 하지 않았을 경우 모든 탭이 화면에 표시됩니다.

 

Parameters

0

편집(Edit) 탭

1

HTML 탭

2

미리보기(Preview) 탭

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.CreateTab = "0|1";
CrossEditor.EditorStart();
