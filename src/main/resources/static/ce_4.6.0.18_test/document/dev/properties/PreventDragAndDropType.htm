PreventDragAndDropTypev4.3.4.05 or higher

파일 종류에 따라 Drag&Drop 기능을 해제 설정합니다.

- 복수 파일 Drag&Drop 시 동영상 파일이 포함된 경우 동영상은 파일로 인식되어 처리됩니다.

- 툴바에서 그림 넣기, 파일 넣기를 삭제한 경우 Drag&Drop 기능도 해제됩니다.

 

Parameters

Drag&Drop 허용 (기본 값)

image

이미지 Drag&Drop 해제

file

파일 Drag&Drop 해제

movie

동영상 Drag&Drop 해제

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PreventDragAndDropType = ["image","movie"];
CrossEditor.EditorStart();
