Method

Event

 

 

CheckNbspInPTagv2.0.5.15 or higher

편집 창에 붙여넣기 시 비어 있는 P 태그에 공백(&nbsp;)이 삽입되도록 설정합니다.

 

값이 없는 P 태그인 경우 편집모드와 조회 모드가 다르게 보이는 현상이 있어 공백 값을 삽입하여 동일하게 보이도록 설정합니다.

 

Parameters

TRUE

공백 삽입

FALSE

공백 삽입 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.CheckNbspInPTag = true;
CrossEditor.EditorStart();
