ClipBoardClearFormattingv4.2.0.16 or higher

편집 창에 붙여넣기 시 모든 글자 속성 지우기 기능 실행 여부 알람의 On/Off를 설정합니다.

 

[ 붙여넣기 시에 속성이 제거된 상태로 붙여넣기를 진행하시겠습니까? 취소 시에는 일반 붙여넣기가 실행됩니다. ]

확인 버튼을 클릭하면 속성이 제거되며, 취소 버튼을 클릭하면 일반 붙여넣기가 실행됩니다.

 

Parameters

TRUE

메시지 창 실행

FALSE

메시지 창 실행 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ClipBoardClearFormatting = true;
CrossEditor.EditorStart();
