Event

 

 

RemoveTableAlignInIE11v3.5.1.07 or higher

Internet Explorer 11버전의 경우 표 안에 align 속성이 있는 경우 한글이 정상적으로 입력되지 않아 align 속성을 제거합니다.

설정하면 Internet Explorer 11버전인 경우 표 생성 시 정렬 기능이 비활성화되며, 탭 이동 및 값을 Get/Set 하는 경우 table 태그의 align 속성이 제거됩니다.

 

Parameters

TRUE

속성 제거

FALSE

속성 제거 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.RemoveTableAlignInIE11 = true;
CrossEditor.EditorStart();
