Param

Method

Event

 

 

Watermarkv4.6.0.12 or higher

워터마크 이미지를 미리 지정하여 이미지 삽입 시 워터마크가 삽입될 수 있도록 설정합니다.

워터마크의 이미지 주소, 위치, 투명도 및 회전 값을 설정합니다.

 

bmp, gif 확장자의 경우에는 png 확장자로 변경되며, gif 이미지는 움직이지 않는 이미지로 삽입됩니다.

 

Parameters

url

워터마크 이미지 URL주소 (필수 값)

position

워터마크 위치
- 0 : 반복
- 1 : 좌측 상단 (기본 값)
- 2 : 우측 상단
- 3 : 좌측 하단
- 4 : 우측 하단
- 5 : 가운데

opacity

워터마크 투명도
- 기본 값 : 0.5
- 설정 값 : 0.1 ~ 1

rotate

워터마크 각도 조절
- 기본 값 : 0
- 설정 값 : 0 ~ 360도

use

자동 워터마크 사용
- true : 사용
- false : 사용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.Watermark = {
    url : "https://comp.namoeditor.co.kr/ce4/images/title.jpg",
    position : "1",
    opacity : "0.5",
    rotate : "0",
    use : "true"
};
CrossEditor.EditorStart();
