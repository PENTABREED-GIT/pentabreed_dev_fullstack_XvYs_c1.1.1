FontSizeListv3.0 or higher

도구 막대에 표시되는 글자 크기의 종류를 설정합니다.

 

설정값은 콤마(,)를 구분자로 하여 배열 형태로 입력하시면 됩니다.

 

기본 글자 크기 목록

["7.5pt", "8pt", "9pt", "10pt", "11pt", "12pt", "14pt", "16pt", "18pt", "20pt", "22pt", "24pt", "26pt", "28pt", "36pt", "40pt"]

설정

글자 크기 목록 설정

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.FontSizeList = ["8pt","9pt","10pt","11pt"];
CrossEditor.EditorStart();
