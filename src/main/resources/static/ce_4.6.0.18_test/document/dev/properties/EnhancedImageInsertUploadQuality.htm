Param

Method

Event

 

 

EnhancedImageInsertUploadQualityv4.3.2.01 or higher

그림 넣기로 이미지 삽입 시 이미지 품질을 낮추어 용량을 줄일 수 있도록 설정합니다.

다만, 이미지에 따른 품질 조절이 아닌 관리자가 설정한 이미지 품질로 일괄 적용됩니다.

 

UseEnhancedImageInsert API가 true로 설정되어야 적용이 가능합니다.

설정 가능 범위는 0~1 사이의 값으로 기본 값은 0.6이며 범위 이외의 값을 설정하면 기본 값으로 동작합니다.

png, jpeg, gif 포맷의 경우 jpg 포맷으로 변환됩니다.

 

Parameters

FALSE

원본 이미지 품질 유지

0 ~ 1 사이의 값

이미지 품질 설정

0.6

이미지 품질 설정 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UseEnhancedImageInsert = true;
CrossEditor.params.EnhancedImageInsertUploadQuality = 0.6;
CrossEditor.EditorStart();
