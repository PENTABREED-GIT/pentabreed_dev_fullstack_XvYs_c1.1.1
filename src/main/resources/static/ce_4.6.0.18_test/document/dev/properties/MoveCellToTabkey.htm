Event

 

 

MoveCellToTabkeyv4.3.2.21 or higher

표 안에서 탭 키 동작 시 셀 간 이동할 수 있도록 설정합니다.

설정한 경우 문단 들여 쓰기는 들여쓰기 툴바를 이용하여 설정이 가능합니다.

 

Parameters

TRUE

Tab키로 셀 간 이동

FALSE

Tab키로 문단 들여쓰기 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MoveCellToTabkey = true;
CrossEditor.EditorStart();
