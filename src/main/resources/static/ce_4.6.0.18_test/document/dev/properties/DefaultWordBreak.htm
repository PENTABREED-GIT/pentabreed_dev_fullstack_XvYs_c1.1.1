DefaultWordBreakv4.1.0.05 or higher

body, table 속성에 줄바꿈 스타일(word-break:break-all)을 설정합니다.

 

한글의 경우 엔터 값을 입력하지 않아도 창 크기에 맞게 줄바꿈 되어 보이지만 숫자/영어의 경우 띄어쓰기나 엔터 값이 없으면 줄바꿈 없이 계속 입력됩니다.

이는 한글은 2byte, 숫자/영어는 1byte 문자로 처리가 다르기 때문입니다.

동일하게 줄바꿈 되도록 하려면 DefaultWordBreak 설정을 합니다.

 

Parameters

TRUE

설정

FALSE

설정 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.DefaultWordBreak = true;
CrossEditor.EditorStart();
