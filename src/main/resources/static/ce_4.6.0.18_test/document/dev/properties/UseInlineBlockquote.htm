Param

Method

Event

 

 

UseInlineBlockquotev4.6.0.11 or higher

인용 구문 적용 시 inline style로 적용되도록 설정합니다.

설정 시 인용 구문 스타일은 아래의 4가지 종류가 출력됩니다.

- 따옴표, 버티컬 라인, 선, 점선

 

Parameters

TRUE

허용

FALSE

허용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UseInlineBlockquote = true;
CrossEditor.EditorStart();
