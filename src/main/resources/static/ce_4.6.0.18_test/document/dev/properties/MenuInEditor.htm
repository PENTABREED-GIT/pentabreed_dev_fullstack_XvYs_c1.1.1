MenuInEditorv4.0.1.12 or higher

상단 메뉴의 드롭다운 메뉴가 에디터 영역 내에 생성되도록 설정합니다.

 

넥사크로에서 팝업 생성 시 z-index 값이 높게 설정되어 에디터의 메뉴가 가려지는 현상이 발생하는 경우 설정합니다.

 

Parameters

TRUE

에디터 영역 내에 생성

FALSE

최상위 document에 생성 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MenuInEditor = true;
CrossEditor.EditorStart();
