MimeEncodeFileScopev4.5.0.01 or higher

MIME 형식으로 데이터를 얻을 때에 인코딩할 대상에 포함할 파일의 범위를 설정합니다.

 

Parameters

설정

파일 범위

0

로컬(에디터 접속 도메인 주소)에 존재하는 파일만 인코딩 (기본 값)

1

로컬(에디터 접속 도메인 주소)과 원격 파일을 모두 인코딩

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MimeEncodeFileScope = 1;
CrossEditor.EditorStart();
