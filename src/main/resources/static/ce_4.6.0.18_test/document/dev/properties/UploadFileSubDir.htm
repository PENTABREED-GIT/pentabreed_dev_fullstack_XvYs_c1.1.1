Event

 

 

UploadFileSubDirv1.5 or higher

파일 업로드 시 서브폴더 생성 여부를 설정합니다.

자동 생성하는 경우 '000001' 형식으로 폴더명이 생성되며, 하위 폴더는 파일 개수가 100개 이상이 되면 자동으로 생성됩니다.

 

Parameters

TRUE

생성 (기본 값)

FALSE

생성 안 함

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UploadFileSubDir = true;
CrossEditor.EditorStart();
