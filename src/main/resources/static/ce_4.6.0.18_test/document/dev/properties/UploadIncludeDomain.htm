UploadIncludeDomainv4.3.3.03 or higher

이미지 붙여넣기 시 설정한 도메인 주소인 경우 이미지를 업로드하도록 설정합니다.

다수의 도메인 설정이 가능하며 ",(콤마)"를 구분자로 입력하시면 됩니다.

이미지 업로드가 실패한 경우 "붙여넣기한 이미지 업로드에 실패하였습니다" 메시지가 출력되며 실패한 이미지 영역에 No Image가 삽입됩니다.

 

단, 이미지 URL의 Base64 정보를 받아온 후 파일로 변경하기 때문에 CORS(교차 출처 공유)가 발생하는 경우 처리가 불가합니다.
