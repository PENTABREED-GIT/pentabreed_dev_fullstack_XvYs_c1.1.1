RemoveScriptInAttributev4.6.0.11 or higher

알림 창(alert, confirm, prompt) 함수가 실행되지 않도록 설정합니다.

설정하면 알림 창 함수를 탭 이동 시, 작성된 내용 리턴(GetValue, GetBodyValue) 시 제거합니다.

CrossEditor.params.RemoveScriptInAttribute = true;
