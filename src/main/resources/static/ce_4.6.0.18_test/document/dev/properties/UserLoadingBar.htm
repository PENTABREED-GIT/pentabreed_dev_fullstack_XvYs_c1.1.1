Param

Method

Event

 

 

UserLoadingBarv4.3.2.33 or higher

에디터가 로딩될 때 사용자 정의 로딩 바가 표시될 수 있도록 설정합니다.

 

Parameters

target

로딩바가 표시되는 엘리먼트 노드

imagepath

로딩바로 사용할 이미지 경로

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UserLoadingBar = {
    target : document.getElementById("editordiv"), 
    imagepath : "../loading.gif"
};
CrossEditor.EditorStart();
