Param

Method

Event

 

 

Width

에디터의 전체 넓이를 설정합니다.

넓이의 입력은 long 타입으로 입력하며 단위는 픽셀(pixel)입니다.

 

Width 값을 설정하지 않는 경우, 기본 값은 "500px"이며 1.1 버전 이하는 "600px"입니다.

1.1 버전 이후부터는 String Type으로 퍼센트(%) 단위가 지원됩니다.

 

Parameters

설정

너비 설정

설정 안 함

기본 값

 

Sample Codes

[ 픽셀(pixel) ]

var CrossEditor = new NamoSE("test"); 
CrossEditor.params.Width = 800; 
CrossEditor.EditorStart();

 

[ 퍼센트(%) ]

var CrossEditor = new NamoSE("test");
CrossEditor.params.Width = "50%";
CrossEditor.EditorStart();
