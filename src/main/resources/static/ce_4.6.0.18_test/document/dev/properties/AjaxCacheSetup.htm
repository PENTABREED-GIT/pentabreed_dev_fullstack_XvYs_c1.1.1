AjaxCacheSetup

Ajax 통신 시 캐시를 사용할지 여부를 설정합니다.

 

패치 업데이트 시 Ajax 통신으로 데이터를 읽어오는 부분에서 브라우저에 캐시 된 파일의 경우 새로운 데이터로 읽어오지 못하는 문제를 해결하기 위해 통신이 필요한 스크립트 파일에 랜덤 한 변숫값을 추가하여 매번 데이터를 호출하도록 합니다.

다만, 설정을 하더라도 브라우저의 옵션에 따라 영향을 받을 수 있습니다.

 

4.3.3.02버전에서 패치 업데이트 시 모든 스크립트 파일이 캐시를 사용하지 않도록 수정되었습니다.

 

Parameters

TRUE

일반통신 (기본 값)

FALSE

캐시 제거된 통신

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.AjaxCacheSetup = false;
CrossEditor.EditorStart();
