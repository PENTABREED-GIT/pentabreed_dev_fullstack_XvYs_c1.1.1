Param

Method

Event

 

 

SaveWithoutNewlinev4.3.2.04 or higher

에디터에서 작성한 내용을 리턴 시 줄바꿈 문자(캐리지리턴)를 제거하도록 설정합니다.

GetValue, GetBodyValue 함수 호출 시 동작합니다.

 

Parameters

TRUE

줄바꿈 문자 제거

FALSE

줄바꿈 문자 유지 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.SaveWithoutNewline = true;
CrossEditor.EditorStart();
