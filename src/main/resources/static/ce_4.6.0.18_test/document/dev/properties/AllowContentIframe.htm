AllowContentIframev2.0 or higher

에디터 본문 내 iframe 허용 여부를 설정합니다.

 

XSS(Cross-site scripting) 취약점 관련 iframe 태그를 이용한 악성 script 삽입을 차단 설정합니다.

에디터의 탭 이동 시, 작성된 내용 리턴(GetValue, GetBodyValue) 시 iframe 태그를 제거합니다.

 

Parameters

TRUE

iframe 허용 (기본 값)

FALSE

iframe 허용 안 함

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.AllowContentIframe = false;
CrossEditor.EditorStart();
