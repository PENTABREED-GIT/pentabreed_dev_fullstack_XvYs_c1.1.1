IsQuestAutoSavev3.0 or higher

도구 막대에서 새 문서 아이콘 클릭 시 내용 삭제 확인 창의 사용 여부를 설정합니다.

 

[ 새 문서를 실행하시면 작성 중인 글이 삭제됩니다. 계속 진행하시겠습니까? ]

확인 버튼을 클릭하면 모든 내용이 삭제되며, 취소 버튼을 클릭하면 내용이 유지됩니다.

 

Parameters

TRUE

메시지 창 실행

FALSE

메시지 창 실행 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.IsQuestAutoSave = true;
CrossEditor.EditorStart();
