ShowDisplayNoneObjv3.5.0.16 or higher

스타일이 display:none인 경우에도 편집화면에 보이도록 설정합니다.

단, position 값에 따라 보일 때 영향을 받기 때문에 옵션으로 설정이 가능합니다.

 

Parameters

TRUE or 1

사용 (position:absolute인 경우에만 표시)

2

사용 (position값 상관없이 모두 표시)

FALSE

사용 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.ShowDisplayNoneObj = true;
CrossEditor.EditorStart();
