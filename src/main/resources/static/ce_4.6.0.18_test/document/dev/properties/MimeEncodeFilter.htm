MimeEncodeFilterv4.5.0.01 or higher

MIME 형식으로 데이터를 얻을 때에 인코딩할 대상에 포함할 파일의 종류를 설정합니다.

 

여러 종류를 사용할 때에는 파일을 포함하지 않는 경우('0')를 제외한 숫자 값들을 합하여 사용합니다.

예를 들면, css와 object를 대상으로 한다면 '1'+'2' 한 값인 '3'을 입력합니다.

 

Parameters

설정

파일 종류

0

파일을 포함하지 않음

1

image (기본 값)

2

css

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.MimeEncodeFilter = 3;    // image + css의 경우
CrossEditor.EditorStart();
