GetTextValueLengthv4.0 or higher

현재 편집 중인 문서의 전체 내용을 HTML 형식으로 반환한 HTML 중 TEXT만 추출하여 문자 개수를 얻어옵니다.

NodeType이 TEXT인 것만 추출하며 <img src="..."/> 형태의 Node 길이는 제외됩니다.

 

Return Values

TEXT만 추출하여 문자 개수를 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.GetTextValueLength();
   alert(info);
}
