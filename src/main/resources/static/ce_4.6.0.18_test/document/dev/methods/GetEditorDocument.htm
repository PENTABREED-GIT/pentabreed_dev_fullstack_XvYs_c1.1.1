GetEditorDocumentv3.0.0.26 or higher

현재 편집 중인 문서에 접근(access) 하기 위해 필요한 객체를 얻어옵니다.

단, 편집모드 상태에서만 DOM 객체에 접근하여 값을 가져올 수 있습니다.

 

Return Values

객체를 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var doc = CrossEditor.GetEditorDocument("doc");
   var info = doc.getElementById("에디터에 선언된 태그 id값").value;
   alert(info);
}
