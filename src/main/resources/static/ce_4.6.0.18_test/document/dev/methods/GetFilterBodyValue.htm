GetFilterBodyValue

현재 편집 중인 문서의 내용을 전자 문서 유통 필터에서 허용하는 태그와 속성으로 정리하여 body 정보(<body>~</body>)를 HTML 형식으로 얻어옵니다.

 

Return Values

body 정보를 HTML 형식으로 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.GetFilterBodyValue();
   alert(info);
}
