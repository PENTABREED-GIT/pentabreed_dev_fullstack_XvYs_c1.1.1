Param

Method

Event

 

 

GetBodyElementsByTagNamev1.5 or higher

현재 편집 중인 문서의 body 태그 안의 태그 이름에 해당하는 객체를 컬렉션 형식으로 얻어옵니다.

 

Parameters

newVal

컬렉션 형식으로 가져올 태그 이름

 

Return Values

지정된 객체를 컬렉션 형식으로 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var TagList = CrossEditor.GetBodyElementsByTagName("img");
   for (i=0; i<TagList.length; i++){
      alert(TagList[i].src);
   }
}
