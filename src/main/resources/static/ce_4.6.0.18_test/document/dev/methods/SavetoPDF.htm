SavetoPDFv4.6.0.03 or higher

PDF 저장하기 기능을 실행합니다.

 

용지 사이즈와 문서 방향을 설정 가능하며 전달 값이 잘못된 경우 기본 값으로 설정됩니다.

[기본 값]

용지 사이즈 = "A4"

문서 방향 = "P"

 

Parameters

type

규격 : A3, A4, A5, B4, B5

orient

방향 : S(가로), P(세로)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SavetoPDF('B4','S');
}
