SetValue

현재 편집 중인 문서의 전체 내용을 HTML 형식으로 설정합니다.

GetValue 함수를 이용하여 저장한 값은 SetValue 함수로 설정합니다.

 

Parameters

newVal

설정할 HTML 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetValue("<html><head><title>test</title></head><body><table><tr>
<td>NamoEditor 문서입니다.</td></tr></table></body></html>");
}
