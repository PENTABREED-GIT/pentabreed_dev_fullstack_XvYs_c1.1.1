Param

Method

Event

 

 

SetCaretPosByIdv4.0.1.04 or higher

지정된 ID 값으로 커서 위치를 설정합니다.

에디터 최초 로딩 시 값을 설정하는 경우 SetFocus = false(실행 안 함)로 설정해야 설정한 ID 값으로 커서가 위치합니다.

 

Parameters

newVal

커서가 위치할 ID 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.SetFocus = false;
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetCaretPosById("obj_id");
}
