Param

Method

Event

 

 

InsertImage

그림과 그림 설명(title)을 삽입합니다.

 

Parameters

url

이미지 파일 URL

url 값이 없는 경우 false 반환

title

이미지 파일 설명 (title=" " 안의 내용)

title 정보 입력은 사용자의 선택사항

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.InsertImage("http://comp.namoeditor.co.kr/ce4/demo/img4/top_ci.png","namo");
}
