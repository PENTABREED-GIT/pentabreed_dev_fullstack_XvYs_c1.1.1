InsertHyperlink

현재 커서가 위치한 곳에 하이퍼링크를 삽입합니다.

 

Parameters

str

문서에 표시할 문자열

입력 커서가 하이퍼링크에 있거나,
블록이 설정되어 있을 경우에는 str 은 무시되며,
그 외의 경우에만 적용됩니다.
또한 str이 없는 경우는 href 가 사용됩니다.

href

하이퍼링크 주소

target

하이퍼링크 대상 프레임 인덱스(0~4)

0

_blank

1

없음

2

_parent

3

_self

4

_top

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.InsertHyperlink("Namo","http://www.namoeditor.co.kr", 1);
}
