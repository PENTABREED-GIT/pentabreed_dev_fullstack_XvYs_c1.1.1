Param

Method

Event

 

 

IsDirty

현재 편집 중인 문서의 내용이 변경되었는지를 확인합니다.

 

Return Values

TRUE

문서의 내용이 변경됨

FALSE

문서의 내용이 변경되지 않음

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetDirty();
}

function SaveContents() {
   var info = CrossEditor.IsDirty();
   if(info)
   {
       alert("내용이 변경되었습니다.");
   }
   else
   {
       alert("변경된 사항이 없습니다.");
   }
}
