Param

Method

Event

 

 

GetHeadValue

현재 편집 중인 문서의 HEAD 정보(<head>~</head>)를 얻어옵니다.

 

Return Values

HEAD 정보를 문자열로 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.GetHeadValue();
   alert(info);
}
