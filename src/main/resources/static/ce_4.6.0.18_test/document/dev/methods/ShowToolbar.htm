ShowToolbar

각 도구 막대(Toolbar)를 보이거나 감춥니다.

 

Parameters

index

도구 막대(Toolbar)의 식별 번호

각 도구 막대의 식별 번호는 '0'부터 시작하여 순서대로 증가합니다.

index에 지정한 도구 막대(Toolbar)가 없을 경우에는 무시합니다.

show

도구 막대(Toolbar)의 표시 여부

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.ShowToolbar(0,1);
   CrossEditor.ShowToolbar(1,0);
}
