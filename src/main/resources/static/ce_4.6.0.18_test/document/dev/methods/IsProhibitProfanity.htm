Param

Method

Event

 

 

IsProhibitProfanity

현재 편집 중인 문서의 내용 중 욕설 관련된 단어가 존재하는지를 확인합니다.

 

Return Values

TRUE

욕설에 관련된 단어 존재

FALSE

욕설에 관련된 단어 미존재

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.IsProhibitProfanity();
   if(info)
   {
       var arr = CrossEditor.GetProhibitProfanity();
       alert("욕설단어가 발견되었습니다." + "\n [" + arr.toString() + "]");
   }
   else
   {
       alert("욕설단어가 발견되지 않았습니다.");
   }
}
