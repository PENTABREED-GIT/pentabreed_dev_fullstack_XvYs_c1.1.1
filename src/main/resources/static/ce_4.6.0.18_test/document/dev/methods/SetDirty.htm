Param

Method

Event

 

 

SetDirtyv1.1 or higher

현재 문서의 편집 상태를 초기화(False) 합니다.

 

에디터를 실행한 후 변경하지 않은 상태로 설정하고 싶은 경우 사용합니다.

SetDirty 실행 시 현재 편집 중인 문서가 초기 문서로 설정이 되어 이후 문서의 변경 유무를 IsDirty 함수를 통하여 확인할 수 있습니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetDirty();
}

function SaveContents() {
   var info = CrossEditor.IsDirty();
   if(info)
   {
       alert("내용이 변경되었습니다.");
   }
   else
   {
       alert("변경된 사항이 없습니다.");
   }
}
