Param

Method

Event

 

 

SetHeadValuev3.0 or higher

현재 편집 중인 문서의 HEAD 정보(<head>~</head>)를 설정합니다.

 

Parameters

newVal

HEAD 정보에 설정할 HTML 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetHeadValue('<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
   <style type="text/css" id="NamoSE__GeneralStyle"> body{font-family :굴림; color : #000000; 
   font-size : 10pt; word-Wrap : break-word; margin: 7px 7px 0 7px;} 
   p,li{line-height:1.2; word-wrap: break-word; margin-top:0; margin-bottom:0;} 
   body{overflow:auto;} .NamoSE_layoutlock_show { word-break: break-all;}
   </style>');
}
