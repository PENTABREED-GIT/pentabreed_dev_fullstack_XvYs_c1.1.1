InsertValue

현재 편집 중인 문서의 특정 위치에 HTML 형식의 내용을 삽입합니다.

 

Parameters

position

내용을 삽입할 위치

0

문서 맨 앞

-1

문서 맨 뒤

1

현재 커서 위치

position 값을 설정하지 않거나, 지정한 위치 값과 다른 값을 입력했을 경우
현재 커서 위치에 내용이 삽입됩니다.

newVal

삽입할 HTML 형식의 내용

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.InsertValue(1, "<p style='color:red;'>Test</p>");
}
