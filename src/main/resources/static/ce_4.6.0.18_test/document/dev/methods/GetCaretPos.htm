GetCaretPosv3.5.1.01 or higher

현재 입력 캐럿의 위치 얻어옵니다.

GetCaretPos로 구한 캐럿의 위치 값은 SetCaretPos에서 사용할 수 있는 값으로 이 값의 절댓값은 아무런 의미를 가지고 있지 않습니다.

 

Return Values

현재 입력 캐럿의 위치 값을 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.GetCaretPos();
   alert(info);
}
