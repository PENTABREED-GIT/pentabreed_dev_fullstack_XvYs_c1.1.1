Param

Method

Event

 

 

SetUISizev 1.1 higher

에디터가 실행된 이후 에디터의 너비와 높이를 동적으로 변경합니다.

 

Parameters

newWidth

변경할 에디터의 너비

newHeight

변경할 에디터의 높이

 

Sample Codes

[ 너비만 변경할 경우 ]

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetUISize("500","");
}

 

[ 높이만 변경할 경우 ]

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e){
   CrossEditor.SetUISize("","600");
}

 

[ 너비와 높이 모두 변경할 경우 ]

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetUISize("500","600");
}
