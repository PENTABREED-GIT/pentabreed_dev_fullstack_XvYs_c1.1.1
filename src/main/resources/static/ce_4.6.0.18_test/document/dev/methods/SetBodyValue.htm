SetBodyValue

현재 편집 중인 문서의 BODY 정보(<body>~</body>)를 설정합니다.

GetBodyValue 함수를 이용하여 저장한 값은 SetBodyValue 함수로 설정합니다.

 

Parameters

newVal

BODY 정보에 설정할 HTML 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   CrossEditor.SetBodyValue("<p>SetBodyValue 값 test 입니다.</p>");
}
