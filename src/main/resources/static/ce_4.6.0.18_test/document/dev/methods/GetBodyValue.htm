GetBodyValue

현재 편집 중인 문서의 BODY 정보(<body>~</body>)를 HTML 형식으로 얻어옵니다.

값을 설정하지 않은 경우 기본 값인 'html'로 설정됩니다.

 

Parameters

HTML

HTML 문법으로 반환 (기본 값)

XHTML

XHTML 문법으로 반환

 

Return Values

BODY 정보를 HTML 형식으로 반환합니다.

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.EditorStart();

function OnInitCompleted(e) {
   var info = CrossEditor.GetBodyValue("XHTML");
   alert(info);
}
